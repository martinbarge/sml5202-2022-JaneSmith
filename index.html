It includes a bootstrap Carousel, which is a great way to add imagery to your website.

When using a carousel, you will have to ensure your images are all scaled to the same aspect ratio.

For example, let's say you choose a width x height ratio of 4 x 2, then, you can use images that are 800px x 400px, or 1000px x 500px, 1200 x 600 etc. You will need to crop your images accordingly to obtain the correct ratios.
